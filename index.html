Tentang Saya

Perkenalkan, saya Raditya Putra Loviartha, seorang Full-Stack Developer dengan semangat tinggi dan dasar yang kokoh dari pendidikan di SMKN 5 Surakarta. Pendidikan vokasi yang saya tempuh telah membekali saya dengan pemahaman praktis dan menyeluruh dalam siklus pengembangan aplikasi, baik untuk web maupun mobile.

Saya memiliki keahlian dalam pengembangan front-end, termasuk penguasaan HTML, CSS, dan JavaScript, serta ReactJS untuk menciptakan antarmuka pengguna yang menarik dan responsif. Di sisi back-end, saya kompeten dalam menggunakan bahasa pemrograman seperti PHP dan Node.js dengan framework seperti Laravel dan Express. Selain itu, saya juga memiliki pemahaman yang baik tentang pengelolaan database MySQL.

Selain pengembangan web, saya juga memiliki keahlian dalam pengembangan aplikasi mobile. Saya familiar dengan Flutter dan Java (Android) serta memiliki pengalaman dalam membangun aplikasi yang berjalan di platform Android. Saya memahami prinsip-prinsip desain antarmuka dan pengalaman pengguna (UI/UX) untuk perangkat mobile.

Dengan pengalaman dalam pengembangan web dan mobile, saya bersemangat untuk terus belajar dan menghadapi tantangan baru dalam dunia teknologi. Saya termotivasi untuk berkontribusi dalam tim yang dinamis dan menerapkan keahlian saya dalam proyek-proyek yang inovatif dan berdampak positif. Saya siap untuk mengaplikasikan pengetahuan dan keterampilan full-stack saya guna menghasilkan solusi aplikasi yang berkualitas tinggi di berbagai platform.
